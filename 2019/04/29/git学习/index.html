进公司实习之前打算恶补一下基础操作

作为一个小萌新进公司实习很慌张啊，偷偷摸摸补点课.

一般而言git操作分为几个关键步骤
1）在本地创建一个仓库 并 git init
2）然后将仓库 git remote add origin git@…… 和远程仓库关联
3）关联之后从远程仓库pull到本地 保持 两仓库一致
4）也可以 将项目文件add到本地 然后提交修改之后 push到远程
